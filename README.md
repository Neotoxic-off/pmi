# PMI
Pimp my intra

### Install:
```
git clone https://github.com/Neotoxic-off/pmi
cd pmi
chmod +x PMI
```

### Requirement
- Stylus (firefox): https://addons.mozilla.org/en-US/firefox/addon/styl-us/

### Usage:
```
./PMI <custom output name>
```

### Custom:
- [X] : Profile Image
- [X] : Main Color
- [X] : Header image

If you need help: [man pmi](https://github.com/Neotoxic-off/pmi/blob/master/DOCUMENTATION.md)
